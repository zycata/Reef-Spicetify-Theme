## Manual
1. Make sure [spicetify](https://spicetify.app/) is installed
2. Download or clone the repo
3. Place it (unzipped) in spicetify themes's directory
    - `%appdata%\Local\spicetify\Themes` on Windows 
    - `~/.config/spicetify/Themes` on Linux 
4. Run `spicetify config current_theme Reef-Spicetify` (or the name of the dir you chose when unzipping)
5. Run `spicetify apply`
6. Change sub-theme by running `spicetify config color_scheme SUB_THEME_NAME`
